# 影核 AI

影核 AI 是一个本地优先的超级智能照片、视频整理与 AI 修图软件。

最终目标：把本地智能相册、自动整理、自然语言找图、AI 修图、批量处理、视频素材管理和企业视觉资产库整合成一个面向个人与企业的视觉资产大脑。

## 当前阶段

已完成：

1. 阶段 1：项目文档与 AI Agent 工作规则。
2. 阶段 2：Tauri + React + TypeScript 桌面空壳。
3. 阶段 3：本地媒体导入第一版。
4. 阶段 4：EXIF 元数据基础能力。

阶段 3 已实现本地文件夹选择、图片/视频递归扫描、SQLite 入库、基础媒体库展示和文件名搜索。
阶段 4 已实现本地照片 EXIF 读取、SQLite 字段入库、拍摄时间与相机型号最小展示，以及按拍摄年份和相机型号搜索。

## 启动桌面应用

前置要求：Node.js、npm、Rust、Tauri 所需系统依赖。

```bash
cd apps/desktop
npm install
npm run tauri dev
```

如果只想看前端页面：

```bash
cd apps/desktop
npm install
npm run dev
```

注意：阶段 3 的“选择文件夹”和“SQLite 入库”依赖 Tauri 桌面环境，单独运行 `npm run dev` 只能查看前端页面。

## 阶段 3 可见内容

桌面窗口应显示：

- 左侧导航栏：全部照片、全部视频、智能相册、AI 修图、批量处理、设置。
- 顶部搜索框：搜索照片、视频、人物、地点或产品。
- 导入文件夹按钮。
- 未导入时显示空状态。
- 导入后显示本地媒体库统计和媒体占位卡片。
- 底部状态栏显示当前状态、总媒体数量、照片数量、视频数量和 AI 状态。

## 阶段 3 已实现功能

- 打开本地文件夹选择框。
- 递归扫描图片和视频文件。
- 支持 jpg、jpeg、png、webp、heic、heif、tif、tiff、bmp、gif、dng、arw、cr2、cr3、nef、orf、rw2、raf、pef、mp4、mov、m4v、avi、mkv、webm。
- 将媒体路径、文件名、扩展名、类型、大小、创建时间、修改时间、导入时间写入本地 SQLite。
- 跳过已经入库的同一路径文件。
- 展示最近 500 个媒体文件的占位卡片。
- 支持按文件名、扩展名、路径和媒体类型做基础搜索。

## 阶段 4 已实现功能

- 导入本地照片时，尽量读取 EXIF 元数据；读取失败时跳过，不阻断导入。
- SQLite 增加拍摄时间、相机品牌、相机型号、镜头信息、GPS 纬度和 GPS 经度字段。
- 媒体卡片展示拍摄时间和相机型号；GPS 只入库，暂不做地图。
- 搜索支持拍摄年份和相机型号。
- 使用 `kamadak-exif` 读取 EXIF。该依赖为纯 Rust 轻量库，不调用外部云服务。

阶段 4 限制：

- EXIF 拍摄时间按照片内原始文本保存，不自行推断时区。
- 部分 RAW、HEIC 或损坏文件可能读不到 EXIF，但不会影响导入。
- 不上传照片，不删除、不移动、不重命名原图。
