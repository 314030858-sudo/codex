# 产品需求文档 PRD

## 产品名称

影核 AI

## 一句话定位

本地优先的超级智能照片、视频整理与 AI 修图软件。

## 产品愿景

让个人、家庭、摄影师和企业拥有一个本地视觉资产大脑：自动整理照片和视频，一句话找图，一键修图，批量处理，并逐步发展为企业级素材资产库。

## 目标用户

### 第一优先级

企业 / 品牌方 / 个人 IP 用户。

核心需求：

- 管理产品图、活动图、会议图、创始人 IP 图和视频素材。
- 快速找到可发朋友圈、公众号、视频号、海报的素材。
- 批量基础修图和未来产品图精修。

### 第二优先级

普通家庭用户。

核心需求：

- 自动整理家庭照片和视频。
- 去重、找废片、找人物、找地点。
- 本地保存，保护隐私。

### 第三优先级

摄影工作室。

核心需求：

- 批量选片。
- 批量人像精修。
- 项目化交付。

## 核心场景

1. 导入一个本地文件夹，软件自动建立照片和视频库。
2. 自动按时间、类型、人物、地点、事件、产品、截图、废片整理。
3. 用户输入自然语言，例如“找会议合影”“找营养品产品图”，系统返回匹配照片。
4. 用户一键基础修图，批量导出。
5. 后续支持人像精修、产品图精修和视频关键帧整理。

## MVP 范围

第一版最小可用产品必须做到：

- 桌面应用能启动。
- 能选择本地文件夹。
- 能扫描照片和视频。
- 能写入 SQLite。
- 能生成缩略图。
- 能在网格展示媒体。
- 能做基础搜索。

## 非目标范围

MVP 暂不做：

- 云同步。
- 商业级人像精修。
- 完整视频剪辑。
- 手机 App。
- 多人权限系统。
- 复杂 RAW 工作流。

## 产品原则

- 先相册，后 AI。
- 先整理，后精修。
- 先本地，后可选云。
- 先能用，后最牛逼。
