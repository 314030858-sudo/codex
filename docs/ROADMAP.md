# 路线图

## 总路线

影核 AI 按阶段推进，避免一次性做复杂大系统。

## 阶段 1：项目文档 ✅

目标：建立项目大脑。

已完成：

- README.md
- AGENTS.md
- docs/PRD.md
- docs/ARCHITECTURE.md
- docs/ROADMAP.md
- docs/MVP.md
- docs/DECISIONS.md

## 阶段 2：桌面空壳 ✅

目标：让桌面应用能启动，有基础界面。

已完成：

- Tauri + React + TypeScript 项目骨架。
- 左侧导航栏。
- 顶部搜索框。
- 导入文件夹按钮占位。
- 空状态页面。
- 能力卡片。
- 底部状态栏。

## 阶段 3：本地媒体导入 ✅

目标：选择本地文件夹，扫描照片和视频，写入 SQLite。

已完成：

- 文件夹选择。
- 递归扫描。
- 识别常见图片、RAW 和视频格式。
- 写入本地 SQLite。
- 跳过同一路径重复导入。
- 显示本地媒体库统计。
- 在网格中显示媒体占位卡片。
- 基础搜索：文件名、扩展名、路径、媒体类型。

未包含：

- 缩略图生成。
- EXIF / GPS 元数据。
- 相似图去重。
- AI 分析。

## 阶段 4：缩略图系统

目标：生成并缓存缩略图，照片网格显示真实图片。

计划功能：

- 为图片生成 256px 缩略图。
- 为视频生成封面占位或后续接入 ffprobe / ffmpeg。
- 缩略图写入缓存目录。
- 数据库记录 thumbnail_path。
- 网格显示真实缩略图，而不是占位卡片。

## 阶段 5：智能相册第一版

目标：自动生成全部照片、全部视频、截图、大文件、可能重复等相册。

## 阶段 6：搜索第一版

目标：支持文件名、日期、类型、扩展名搜索，并预留 AI 搜索接口。

## 阶段 7：AI Worker 第一版

目标：建立 Python AI Worker 和任务队列，不急着接大模型。

## 阶段 8：自然语言搜图

目标：接入本地 CLIP 或兼容模型，实现语义搜索。

## 阶段 9：基础非破坏式修图

目标：曝光、对比度、饱和度、色温、锐化、导出。

## 阶段 10：人像精修

目标：轻度磨皮、祛痘、肤色均衡、眼部增强。

## 阶段 11：视频整理

目标：视频封面、元数据、关键帧、后续视频 AI 标签。
