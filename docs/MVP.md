# MVP 定义

## MVP 目标

第一版最小可用产品的目标是：

> 能打开软件，能选择本地文件夹，能扫描照片和视频，能写入 SQLite，能生成缩略图，能在网格展示，能做基础搜索。

## MVP 必做功能

1. 桌面应用启动。
2. 文件夹选择。
3. 图片和视频识别。
4. SQLite 入库。
5. 缩略图生成。
6. 媒体网格展示。
7. 基础搜索。
8. 本地优先隐私说明。

## MVP 暂不做

- 人像精修。
- 产品图高级精修。
- 视频自动剪辑。
- 云同步。
- 手机 App。
- 多人权限。
- 私人审美模型。

## 阶段 2 状态

阶段 2 已完成桌面空壳。

阶段 2 只验证：

- 窗口能启动。
- UI 布局方向正确。
- 用户能看到导入入口和未来能力。

阶段 2 不验证真实媒体功能。
